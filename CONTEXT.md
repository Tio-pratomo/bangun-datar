# Context

## Tech Stack
- Vite
- React
- Tailwind CSS (v4)

## Glossary

### Bangun Datar
Objek 2D. Hitung Luas, Keliling.
Daftar: Persegi, Persegi Panjang, Segitiga, Lingkaran, Trapesium, Jajar Genjang, Belah Ketupat, Layang-layang.

### Bangun Ruang
Objek 3D. Hitung Volume, Luas Permukaan.
Daftar: Kubus, Balok, Tabung, Kerucut, Bola, Limas (Segitiga, Segiempat, Segilima, Segienam), Prisma (Segitiga, Segiempat, Segilima, Segienam).

### Konversi Satuan
Ubah nilai antar satuan.
Jenis: Panjang (km–mm), Volume (kl–ml), Berat (kg–mg).

### Arsitektur
Halaman dinamis tunggal (SPA). Konten ganti via React state.

### Features
- Dark Mode support
- Perhitungan Bangun Datar (Luas, Keliling)
- Perhitungan Bangun Ruang (Volume, Luas Permukaan)
- Konversi Satuan (Panjang, Volume, Berat)
