# ADR 0004: Tailwind CSS untuk Styling

## Status
Accepted

## Context
Butuh styling cepat, konsisten, dan responsif.

## Decision
Gunakan Tailwind CSS v4 via @tailwindcss/vite plugin.

## Consequences
Styling via utility classes. Tidak perlu PostCSS. Konfigurasi via CSS import `@import "tailwindcss"`.
